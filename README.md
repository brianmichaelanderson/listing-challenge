# Property Listing Flow - Coding Challenge

## Quick Start

```bash
# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Start development server
npm run dev
```

Open [http://localhost:3032/listing-flow/step-1](http://localhost:3032/listing-flow/step-1)

## Instructions

Please see **TASK.md** for the complete challenge instructions and requirements.

## Project Structure

```
src/
├── app/
│   ├── api/              # API routes
│   └── listing-flow/     # Page routes
├── sections/             # Business logic components
├── components/           # Reusable UI components
├── lib/                  # Utilities (axios, mock supabase)
├── auth/                 # Auth context
└── theme/                # MUI theme
```

## Stack

- Next.js 15 (App Router)
- React 19
- Material-UI v6
- React Hook Form + Zod
- Axios
- Mock Supabase (no real database needed)
